# ppt-image-blur-folding-effect

BV1Ka5X6AEax 视频中PPT效果的批量应用。

## 简介

将 `images/` 目录下的图片批量处理，生成带有毛玻璃模糊背景 + 折叠页角效果的 PPTX 演示文稿。每张图片生成一页幻灯片，包含：

- 全幅毛玻璃模糊背景（图片的模糊版本 + PowerPoint 原生玻璃艺术效果）
- 右侧折叠裁剪的原始图片（使用自定义几何路径裁剪）
- 折叠边缘的装饰性渐变三角形（模拟3D光照效果）

## 使用方法

```bash
# 1. 将图片放入 images/ 目录
# 2. 确保 sample.pptx 存在于项目根目录
# 3. 运行脚本
python main.py
# 4. 生成的 PPTX 保存在 output/output.pptx
```

### 依赖

```
python-pptx pillow numpy lxml
```

## 效果原理

脚本复现了以下 PowerPoint 手动操作流程：

1. **模糊**：对原图应用高斯模糊（Pillow `GaussianBlur(sigma=15.0)`，等效于 PPT 艺术模糊半径 30）
2. **玻璃效果**：对模糊后的图片应用基于**正弦波置换映射（Displacement Mapping）**的像素扭曲。PPT 的"艺术玻璃"效果本质上是让像素有规律地"搬家"——通过正弦波函数为每个像素计算偏移量 `(dx, dy)`，然后在新位置重采样原图像素，模拟透过有纹理的玻璃观看时的折射偏移感。参数：`GLASS_SCALE=34`（对应 PPT 缩放，控制扭曲振幅 = scale/5），`GLASS_TRANSPARENCY=0`（对应 PPT 透明度 0%）
3. **折叠裁剪**：将原图嵌入 PPTX，通过 `<a:custGeom>` 自定义几何路径（含三次贝塞尔曲线）裁剪为折叠页角形状
4. **装饰三角形**：两个带渐变填充的三角形（`accent1` 蓝色 → `bg1` 白色），垂直翻转后组合，模拟折叠处的3D光影

## 作者发给智能体的提示词

> @sample.pptx 中，你通过手工操作把第2页的素材变成了第1页的模样：
> 1. 创建一个形状不太一样的三角形"单个下图形"
> 2. 你把这个图形复制一份，上下反转，然后让"下图形"和"上图形"组合形成"图形组合"
> 3. 你把"图形组合"的高度调整了一下，确保"图形组合"刚好占满PPT页面的高度
> 4. 你复制了一份"原图"，得到"模糊图片"，设置艺术特效为模糊，半径30
> 5. 你复制了一份"模糊图片"，然后在粘贴选项里选择"粘贴图片"，粘贴出了一个没有艺术效果的模糊图片"模糊玻璃图片"
> 6. 你给"模糊玻璃图片"添加了玻璃艺术效果，透明度0%，缩放34，设置了16:9裁剪
> 7. 你绘制了一个多边形"多边形"
> 8. 你把"原图"和"多边形"都贴着幻灯片页面的右边摆放，高度和幻灯片页面相同。对于其它的元素，照着第1页的位置、大小和前后摆放
> 9. 你先选中"原图"，然后选中了"多边形"，用合并图形的相交，把原图裁剪成了多边形的模样
>
> 最后，你得到了第1页的效果。现在你要创建脚本自动化这个流程，把 @images\ 的图片都处理一下，得到这种效果，把生成好的PPT放到 @output\ 里。
> 上边的操作是直接在PowerPoint上完成的，你用脚本效果可能不一样，必须调查清楚每一步的原理，以及用脚本等效实现的方法。
